# OpenDesign Prototype Reference

These files are copied from the OpenDesign prototype export and are kept as implementation references, not app source.

Useful entry points:

- `widget-main.html`: primary desktop widget direction.
- `settings-widget.html`: widget appearance settings reference.
- `settings-accounts.html`: account connection settings reference.
- `styles/widget.css`: widget visual system reference.
- `styles/settings.css`: settings visual system reference.
- `scripts/widget.js`: prototype interactions such as view switching, drawer toggle, and drag/drop.
- `scripts/settings.js`: prototype settings interactions.

Notes:

- Screenshots from the OpenDesign discussion are intentionally not kept here because they were used for communication and correction rather than implementation reference.
- Deprecated OpenDesign pages such as the exported `index.html`, `widget-empty.html`, and `widget-busy.html` are intentionally not kept here.
- Use these files to guide React/Tauri implementation, but keep production source under `code/`.
